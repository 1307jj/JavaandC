# StudentManager
학부생/대학원생 성적 관리 프로그램입니다.

주요 기능
- 학생 추가 / 삭제 / 검색
- 전체 성적 조회
- 통계 (평균, 최고점, 최저점, 학점 분포)
- 석차 조회

사용 개념
- 상속, 다형성 (Ch6, Ch7)
- 사용자 정의 예외처리 (Ch8)
- String 처리 (Ch9)
